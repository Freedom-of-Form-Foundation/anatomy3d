Detailed Description

A ContinuousMap is an abstract class that defines some function that outputs a value of type TOut for each input of type TIn. Mathematically, this means it defines a mapping from one space to another. For example, a ContinuousMap<Real, Real> defines a function that takes a Real value and outputs a Real value, such as a polynomial function. ContinuousMap<Vector2, Real> takes a Vector2 as input and gives a Real as output, which could for instance be used as a height map.
