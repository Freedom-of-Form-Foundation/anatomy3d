Detailed Description

A SortedPointsList stores points in Lists so that BinarySearch can be used. The points in this list are guaranteed to be sorted by Location-value and to have no duplicate entries for the Location-coordinate. This is a bit roundabout, and should probably be replaced once a BinarySearch is added for IList or SortedList.
