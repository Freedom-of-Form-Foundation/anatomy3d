Solves the equation \((q(x))^2 = b_0 + b_1 x + b_2 x^2 + b_3 x^3 + b_4 x^4\), returning all values of \(x\) for which the equation is true. \(q(x)\) is the continuous map. The parameters z0 and c can be used to substitute x, such that \(x = z0 + c t\). This is useful for raytracing.
