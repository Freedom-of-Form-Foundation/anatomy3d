Construct a Slice viewing a specific list at a specific range.

Slice performs bounds-checking in debug builds, but skips it in release builds. It is intended to be used as a higher-performance alternative to GetRange when the original list will not change and no copy is explicitly desired.

Parameters
listList to slice.
offsetThe index of the original list that should correspond to the 0th index of the slice; the first index that is in range (if the slice has a length greater than 0).
countThe number of items to include in the slice. The last included item of the underlying list will be the item at index offset + count - 1.
